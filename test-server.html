로컬 서버 실행 방법

방법 1: Python 사용 (권장)

터미널에서 다음 명령어를 실행하세요:

python -m http.server 8000

그 다음 브라우저에서 http://localhost:8000으로 접속하세요.

방법 2: Node.js 사용

터미널에서 다음 명령어를 실행하세요:

npx http-server -p 8000

그 다음 브라우저에서 http://localhost:8000으로 접속하세요.

방법 3: VS Code 사용

VS Code에서 "Live Server" 확장을 설치하고, index.html을 우클릭 → "Open with Live Server"를 선택하세요.

# 블로그에 오신 것을 환영합니다! 👋

안녕하세요! 이것은 GitHub Pages로 호스팅되는 정적 블로그의 첫 번째 게시글입니다.

## 이 블로그의 특징

이 블로그는 다음과 같은 기능을 지원합니다:

- **마크다운 기반 글쓰기** - 간편하게 마크다운으로 글을 작성하세요
- **다크/라이트 모드** - 오른쪽 상단의 버튼으로 테마를 전환할 수 있습니다
- **코드 하이라이팅** - 다양한 프로그래밍 언어의 문법 강조
- **태그 필터링** - 태그별로 게시글을 필터링할 수 있습니다
- **검색 기능** - 게시글 제목과 내용을 검색할 수 있습니다
- **Giscus 댓글** - GitHub Discussions 기반의 댓글 시스템

## 코드 예시

JavaScript로 "Hello, World!"를 출력하는 예시입니다:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

Python 예시도 살펴볼까요?

```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

## 새 게시글 작성하기

`pages/` 폴더에 새로운 `.md` 파일을 만들면 됩니다:

1. `pages/` 폴더에 `your-post-title.md` 파일 생성
2. 파일 상단에 Front Matter 작성:

```markdown
---
title: '게시글 제목'
date: 2025-01-26
tags: ['태그1', '태그2']
category: '카테고리'
description: '게시글 설명'
---

여기에 내용을 작성하세요...
```

3. `git push`하면 자동으로 배포됩니다!
